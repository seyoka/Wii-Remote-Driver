# Wii Remote HID Bluetooth Driver

## Project Overview
This project is a **custom Linux device driver** for the **Nintendo Wii Remote**, enabling it to function as an **HID (Human Interface Device) over Bluetooth**. The driver is developed for the **ISE 4422A - Operating Systems Project** at the University of Limerick.

## Features
- ✅ Supports Bluetooth communication with the Wii Remote
- ✅ Implements HID input mapping for button presses
- ✅ Handles device pairing and connection management
- ✅ Processes HID reports for accelerometer and motion control
